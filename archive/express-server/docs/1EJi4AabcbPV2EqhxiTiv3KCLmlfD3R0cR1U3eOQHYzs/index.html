壹、慈濟歌曲：〈掘井人〉

報告人：文史處 江淑怡

自我期許掘井人

一九七九年，上人發起籌建「佛教慈濟綜合醫院」的構想後，有一天陳燦暉教授來訪，除了肯定上人的大願，也憂心上人這個願真的能達成嗎？陳教授表示，建院構想緊隨而來的，將是一連串的艱辛困苦。

不畏艱難掘井人

  • 上人以堅定的行動回應眾人的質疑與勸告！
  • 上人知道，要為東部蓋醫院，有多麼不容易；一甲子前，上人以掘井的精神，跨出慈善的腳步，再完成花蓮慈濟醫院的籌建，讓搶救生命的良能可長久發揮。

掘井人啟發眾人行善

一九九一年三月廿三日 新竹交通大學師生參訪座談 上人開示

我舉了一個「掘井人」的例子，例如一個工程師，設計了一個很大的水池；這個水池用磚塊、用水泥、用人工砌起來，任憑它有多大，都要靠上面下來的雨水，一點一滴下來。

水池下面是乾燥的土地，要有很充沛的水，土地才能夠受到滋潤；慈濟會員就像點點滴滴的水，委員就像磚和水泥，我就像設計水池的人。萬一上面的水源斷了，下面的水就無法流出去了。

一九七九年，是我身體最危急的時候。當時說暈過去就暈過去，心絞痛一天不知道要發作幾次，尤其是天將亮的時候，時常都是突然間一陣絞痛。我一直都感覺到人生無常，萬一有一天我一口氣出去不再進來，當時我們所救濟的已有三千多個孤老無依及寡婦孤兒，這些人要怎麼辦呢？

人家說眼不見為淨，不過我已經看到了，叫我怎麼忍心呢？蓋蓄水池的功夫倒不如來挖井，這個井要一直掘到湧出泉來，到時候我就不必煩惱。

掘井人啟發眾人大愛

所以我還是不斷地在掘井！我們把醫院交給院長，交給醫生，把慈濟的精神灌輸給他們，他們就可以發揮愛的良能。醫生做病人心目中的活佛，護士成為病人心目中的活菩薩。

期許弟子都是掘井人

二〇一九年十二月二十日 與高雄區慈濟人座談開示

一缸水，供應一杯一杯，止渴；還要再供應更多人，就要整條溪水、河水、池水來供應；要再供應更多人，那就要出泉水了。人人的法入心，啟發了心泉，智慧之泉源就會不斷湧出。

大家要師父免操心，要師父永遠永遠都在；想要求這個「身」永遠在，是不可能。想要求心靈這一念心，這個湧泉，就要看人人有沒有接受；人人接受了，掘井人，永遠在你們的心靈，愈開湧泉愈大。

身體力行真正掘井人

辛巴威朱金財師兄曾經四度遭搶劫、資產歸零，一念轉變，如今是在辛巴威修建五千口井、造福八百萬人、實實在在的掘井人！他不僅為乾涸的土地引出生命之水，更與本土志工建立親如家人的至深情誼。

這份情誼，正是大愛湧泉之源，他將當地人視為自己的家人，真誠為他們付出。

二〇二五年九月，朱師兄在辛巴威靜思堂動土典禮上說：「當我結束生命，我要葬在這片土地；未來，我會跟你們坐在一起，有一樣的膚色，為辛巴威肝腦塗地。」在哽咽淚水中，他真誠地為今生與來世和辛巴威的情誼下了註腳。

資料來源：第七〇九期慈濟月刊（二〇二五年十二月）

虔敬感恩

貳、鳳山五甲和氣 過埤里社區關懷據點—長情有愛 銀向幸福

二〇二六年一月五日

南部社福室 方信智（本愷） 鳳山五甲和氣 邱芹珍（慈芹）

高雄區據點現況

高雄區自二〇一七年推動與協會、里長共同合辦之外展社區關懷據點，目前共十七個點，鳳山區占十一個。

鳳山區過埤里

鳳山區過埤里位於五甲社區東方，地廣人稀六十五歲長者約千餘人，是五甲新開發地區。

過埤里關懷據點自二〇一七年成立迄今已邁入第九年，學員四十三人。

與里長共善

與慈濟長時間的配合共善，過埤里莫尚忠里長，現在是慈濟的最佳後盾，即便公務繁忙，一定抽空來參加慈濟活動。

課程內容

課程包含預防失智失能的手作課程、健康促進的適性運動、慈濟人文手語帶動等豐富多元課程。

弟子們謹記上人殷殷叮嚀，要我們把社區的父母和兄弟姊妹照顧好。

疫起轉型線上教學

二〇二二年五月因疫情嚴峻停課。面對長者年紀較大，不善使用連線軟體難題，團隊努力克服順利上線教學。

踏出了第一步以後，團隊進而挑戰在線上教學手作小綿羊黏土和蔬食三明治課程。

跨越三千多公里的愛

來自印尼的Sri，三年前她推著坐輪椅的阿嬤一起來據點上課，個性開朗臉上總是掛著笑容。

手腳俐落的她，除了照顧阿嬤，也會幫著同桌的長輩們穿針畫線的，是長輩們的得力助手。長輩們也會幫忙照顧阿嬤。

因工作年限已到，Sri即將返回印尼。最後一次帶阿嬤來上課時，她與長輩、志工緊緊相擁道別。離別的瞬間讓整個據點都濕了眼眶。

據點長輩們在群組留言祝福她。

動態成果發表

七月吉祥月據點祈福活動，長者化身小老師。

運用在課程中學習的內容，帶領會眾動手做艾草槌、陶土茶盤、蔬食沙拉、草頭娃娃、花草茶及種子動物園。

長輩是志工最佳後盾

五甲共修處在十月興建完成。迎來第一次的歲末祝福，志工們結合據點長輩，把場地布置成懷舊風貌。

長輩們歡喜地來當闖關遊戲志工，讓活動圓滿。

據點有愛—文雄爸爸的故事

無常悄悄來敲門

辛文雄爸爸和莊英娥媽媽夫妻檔，是二〇一七年據點開辦第一批的長輩。

據點有愛的地方

英娥媽媽在二〇二四年十二月因心肌梗塞離世，據點志工和學員一起到殯儀館關懷文雄爸爸。

陪伴走出悲慟

女兒們輪流陪伴爸爸一起來上課。

女兒辛宜玲分享

二〇一七年爸媽歡喜地參加過埤關懷據點，而良善種子就慢慢注入。每每上完據點課程後，爸媽就會迫不及待與我分享今天在據點學到什麼，知道什麼新的知識。

在家爸媽會互相幫忙把在據點未完成的事一起完成，或到市場採買食材，歡喜的把剛學會的素食做給家人們享用；爸媽的共同做的事愈來愈多互動更好，他們告訴朋友，希望朋友們也能來據點學習讓年老生活更精采。

二〇二四年十二月突如其來的無常，把親愛的媽媽無聲無息帶離我們身旁，悲慟之餘，家人試著慢慢調整步伐回歸正常生活。我們仍鼓勵爸爸參加每周的據點，因為那裏有爸爸最熟悉像家人般的同學與慈濟師兄姊的陪伴與學習。

大姊也因陪伴爸爸共學的機緣，報名參加許多慈濟志玄課程，大姊也常常與家人分享她中學習的喜悅與觀念的改變。

最後，我要感謝上人的遠見與照顧到社區老人，也感恩志工發現爸爸生活中的亮點，讓他覺得每次走入據點時內心是充滿歡喜，讓年邁的爸爸能再繼續揮灑精彩的人生故事。
